# Number Guessing Game (Python)

## Project Description
This is a simple console-based Number Guessing Game written in Python.  
The computer randomly selects a number between 1 and 100, and the player has to guess it within a limited number of attempts based on the chosen difficulty level.

The project is developed using **Visual Studio Code**, version-controlled using **Git**, and hosted on **GitHub**.

---

## Technologies Used
- **Language:** Python 3
- **IDE:** Visual Studio Code
- **Version Control:** Git & GitHub

---

## How to Run

1. **Clone the repository:**

   git clone https://github.com/devsehgal11/cs-assignment-4.git/
   cd number-guessing-game

2. **Run the game:**

   python guess_game.py
